## 主なデバッグ方法

* VSCodeの「View In Browser」プラグイン
	* HTMLファイルを右クリック -> 「View In Browser」
	* 設定に`"view-in-browser.customBrowser": "chrome"`を追加し、ブラウザの指定が可能

* VSCodeのDebgger for Chromeプラグイン
	* Chromeのみ
